# EdgeFlow Quickstart

Get EdgeFlow running end-to-end in under 5 minutes.

---

## Prerequisites

- Python 3.8 or higher
- Arduino IDE 2.x with ESP32 board package installed
- An ESP32 DevKit board (optional — you can test without hardware)

---

## Step 1 — Install the Python Toolkit

```bash
git clone https://github.com/ajharshal45/Edge-AI-Framework.git edgeflow
cd edgeflow/python
pip install -e .
```

Verify installation:

```bash
edgeflow --version
# EdgeFlow 1.0.0
```

---

## Step 2 — Generate Training Data

EdgeFlow includes a built-in synthetic data generator for the SafeNest domain:

### CLI
```bash
edgeflow generate-data --domain safenest --samples 500 --output safenest_data.csv
```

### Python API
```python
from edgeflow.datasets.builtin.generate_safenest import generate_safenest_dataset

df = generate_safenest_dataset(n_samples=500, output_path="safenest_data.csv")
print(df.head())
```

Output CSV format:
```
mq2_raw,pir_state,label
2742,0,DANGER
1212,1,SAFE
3512,0,DANGER
```

---

## Step 3 — Train the Model

### CLI
```bash
edgeflow train \
  --data safenest_data.csv \
  --model decision_tree \
  --target esp32 \
  --domain safenest
```

### Python API
```python
from edgeflow import EdgeFlow

ef = EdgeFlow(domain="safenest", target="esp32", model="decision_tree")
report = ef.train("safenest_data.csv")

print(f"Accuracy   : {report.accuracy * 100:.1f}%")
print(f"Model size : {report.model_size_kb} KB")
print(f"Label map  : {report.label_map}")
```

Expected output:
```
Accuracy   : 96.4%
Model size : 8.0 KB
Label map  : {'DANGER': 0, 'SAFE': 1}
```

## Step 4 — Validate Device Compatibility

### CLI
```bash
edgeflow validate --target esp32 --model decision_tree
```

### Python API
```python
result = ef.validate_target()
print(f"Compatible: {result.compatible}")
print(f"RAM usage : {result.estimated_ram_kb} KB")
```

Expected output:
```

=======================================

Model           : decision_tree
Estimated RAM   : 3.2 KB
Estimated Flash : 8.1 KB
ESP32 DEVKIT RAM Limit : 320 KB
=======================================
[OK] Compatible -- Safe to deploy
```

## Step 5 — Export to C++ Header

### CLI
```bash
edgeflow export --output safenest_model --format both
```

### Python API
```python
paths = ef.export(output="safenest_model", format="both")
# Generates: safenest_model.h  and  safenest_model.edgeai
```

---

## Step 6 — Flash to ESP32

1. **Copy** `safenest_model.h` into your Arduino sketch folder

2. **Install EdgeFlow C++ library**: copy `cpp/` into your Arduino/libraries folder

3. **Create** or open `SafeNestIntegration.ino`:

```cpp
#include "safenest_model.h"   // generated by EdgeFlow
#include <EdgeFlow.h>

void setup() {
    Serial.begin(115200);
    EdgeFlow.begin();
}

void loop() {
    float input[] = {
        (float)analogRead(34),     // MQ-2 gas sensor
        (float)digitalRead(27)     // PIR motion sensor
    };

    EdgeFlowResult result = EdgeFlow.predict(input, EDGEFLOW_N_FEATURES);

    if (result.label == 0) {   // 0 = DANGER
        // trigger gas alert
        Serial.println("DANGER detected!");
    }

    Serial.print("Latency: "); Serial.print(result.latency_ms); Serial.println(" ms");
    delay(500);
}
```

4. **Upload** and open Serial Monitor at 115200 baud

## What Happens at Each Step

```
[Laptop]                                [ESP32]
   |                                       |
   | generate-data                         |
   |   --> safenest_data.csv               |
   |                                       |
   | train                                 |
   |   --> fitted sklearn model            |
   |                                       |
   | export                 copy via USB   |
   |   --> safenest_model.h  ----------->  |
   |                                       |
   |                              EdgeFlow.predict()
   |                              runs locally
   |                              0 bytes to internet
   |                              < 0.1ms latency
```

---

## Label Map Note

EdgeFlow's LabelEncoder sorts labels alphabetically. For SafeNest:

| Label  | Integer |
|--------|---------|
| DANGER | **0**   |
| SAFE   | **1**   |

Always check `result.label == 0` for DANGER.
