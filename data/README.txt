EdgeAI Framework - Training Data
=================================

This directory holds the training CSV files for each domain:

- healthcare.csv   : Heart rate, BP, temperature, SpO2, activity
- smartcity.csv    : Vehicle count, speed, density, time, weather
- environment.csv  : Temperature, humidity, PM2.5, PM10, CO2

These datasets are used by the trainer module to generate
the pretrained .pkl models that ship with the package.
